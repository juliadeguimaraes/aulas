Principais formatações de texto

Negrito / Destaque

Nessa frase temos um termo em negrito para usar o negrito faça: <b>, uma tag não semântica

Nessa frase temos um termo em destaque usando uma tag semântica <strong>

Itálico / ênfase

Nesta frase temos um termo itálico usando a tag <i> (não semântica)

nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Textos grandes e pequenos

estamos criando um texto grande (o texto grande precisa ser formatado com css) e um texto pequeno usando <small> nesse parágrafo

Texto deletado

Podemos marcar um texto como excluído para indicar que o texto deve ser lido mais não considerado, usando <del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase indicando que ele foi adicionado depois, usando <ins>

Existe também o sublinhado com a tag <u> que é não semântica

Texto sobrescrito

Para inserir coisas do tipo x20+3, use a tag <sup>

Texto subscrito

Para inserir texto H20, use a tag <sub>
